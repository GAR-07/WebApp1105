Web приложение, позволяющее осуществлять вход в аккаунты из списка одним из двух выбранных способов авторизации и аутентификации.
Поддерживается как аутентификация через Cookie, так и через Bearer токен.

API часть проекта открывается через Visual Studio и запускается, в дальнейшем игнорируется и все взаимодействия происходят через UI
UI запускается через консоль командой "ng serve --o", находясь по адрессу папки WebApp1105.UI

Используются:
ASP.NET 6
Angular 16.0.2
bootstrap 5.3.0-alpha1

Для работы проекту нужна база данных, приложение ffmpeg.exe и настройка путей 
для этих компонентов в appsettings.json и StorageController соответственно.
Для экономии места из проекта были исключены папки в Resources, в которых содержатся файлы.
Новые папки будут создаваться автоматически при загрузке файлов.
